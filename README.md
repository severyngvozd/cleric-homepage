# Cleric.ai Homepage

Responsive homepage built with AstroJS.

## Setup

```sh
npm install
npm run dev
```

## Features

- All homepage sections (Hero, Features, Meet Cleric, In Action, Under The Hood, Security, Integrations, CTA, Footer)
- Responsive design (mobile, tablet, desktop)
- Two-page "Try it out" form wizard
- Sticky navigation bar
- Dismissible top banner
- Form submissions to webhook.site

## Build

```sh
npm run build
npm run preview
```

## Deployment

Deploy to Vercel, Netlify, Cloudflare Pages, or any static hosting service.
